# 트라이아지 라벨

엔지니어링 스킬들은 5개의 표준 트라이아지 역할을 기반으로 작동합니다. 이 파일은 해당 역할을 이 프로젝트 이슈 트래커에서 사용하는 실제 라벨 문자열로 매핑합니다.

| mattpocock/skills의 라벨 | 이 프로젝트의 라벨 | 의미                                    |
| ------------------------ | ------------------ | --------------------------------------- |
| `needs-triage`           | `needs-triage`     | 유지보수자가 이슈를 평가해야 함          |
| `needs-info`             | `needs-info`       | 보고자로부터 추가 정보 대기 중           |
| `ready-for-agent`        | `ready-for-agent`  | 완전히 명시됨, 에이전트가 작업 가능      |
| `ready-for-human`        | `ready-for-human`  | 사람의 구현이 필요함                     |
| `wontfix`                | `wontfix`          | 조치하지 않음                           |

스킬이 특정 역할을 언급할 때(예: "AFK-ready 트라이아지 라벨 적용"), 이 테이블의 해당 라벨 문자열을 사용하세요.

오른쪽 열을 실제로 사용하는 용어로 수정하세요.
